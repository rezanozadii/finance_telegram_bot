# ربات تلگرام مدیریت مالی شخصی

یک **دستیار مالی هوشمند** کامل، ساخته‌شده به‌صورت ربات تلگرام + Mini App. درآمد و هزینه‌ها را ردیابی کنید، حساب‌ها را مدیریت کنید، هزینه‌ها را با دوستان تقسیم کنید، و از مربی مالی هوش مصنوعی داخلی بهره‌مند شوید — همه درون تلگرام.

## امکانات

### ربات

#### مالی اصلی
- **حساب‌ها** — نقدی، کارت، بانک، کیف پول الکترونیکی، اعتباری با ردیابی لحظه‌ای موجودی
- **دسته‌بندی‌ها** — ۱۴ دسته‌بندی پیش‌فرض + امکان ایجاد دسته‌بندی سفارشی
- **ثبت دستی تراکنش** — درآمد، هزینه، انتقال با همگام‌سازی موجودی حساب
- **تجزیه‌ی هوش مصنوعی** — یک پیام به‌زبان طبیعی بفرستید («۵۰ هزار تومان برای ناهار پرداختم») و DeepSeek به‌صورت خودکار تراکنش را استخراج می‌کند
- **یادگیری دسته‌بندی** — اصلاح دسته‌بندی پیشنهادی هوش مصنوعی، قوانین مرتبط با همان فروشنده را برای دفعات بعد ذخیره می‌کند
- **تراکنش‌های تکرارشونده** — قبوض هفتگی/ماهانه/سالانه با یادآوری روزانه و تأیید با یک ضربه
- **دوستان و تقسیم هزینه** — ارسال درخواست دوستی، ثبت هزینه‌های مشترک، ردیابی بدهی به تفکیک ارز، تسویه‌حساب
- **گزارش‌ها** — خلاصه‌ی ماهانه / فصلی / سالانه با تفکیک دسته‌بندی و مقایسه با دوره‌ی قبل

#### مربی مالی هوشمند
- **`/ai`** — گفتگوی طبیعی با دستیار مالی هوش مصنوعی («آیا می‌توانم تعطیلات بروم؟»، «کجا بیشتر از حد هزینه می‌کنم؟»)
- **`/health`** — امتیاز سلامت مالی (۰ تا ۱۰۰) بر اساس ۸ مؤلفه‌ی وزن‌دار: نرخ پس‌انداز، ثبات هزینه، رعایت بودجه، جریان نقدی، ثبات درآمد، صندوق اضطراری، نسبت هزینه‌های تکرارشونده، تنوع تراکنش‌ها
- **`/goals`** — تعریف اهداف مالی (لپ‌تاپ، مسافرت، صندوق اضطراری) و پیگیری پیشرفت با تخمین تاریخ تکمیل توسط هوش مصنوعی
- **`/budget`** — تنظیم سقف هزینه ماهانه/هفتگی/سالانه به تفکیک دسته‌بندی با هشدار در ۸۰٪ / ۹۰٪ / ۱۰۰٪
- **`/insights`** — ۳ بینش پیشگیرانه‌ی روزانه که هر روز ساعت ۰۸:۰۰ توسط هوش مصنوعی تولید می‌شود
- **`/coach`** — خلاصه‌ی مربیگری هفتگی با مشاهدات شخصی‌سازی‌شده و توصیه‌ها
- **شخصیت مالی** — طبقه‌بندی خودکار: استاد بودجه، پس‌اندازکننده‌ی متعادل، هزینه‌کننده‌ی سبک زندگی، بدهکار سنگین و ...
- **تشخیص اشتراک‌ها** — شناسایی خودکار پرداخت‌های دوره‌ای با هزینه‌ی ماهانه/سالانه
- **تشخیص عادات** — شناسایی عادت‌های هزینه‌ای مکرر با فرکانس، میانگین و هزینه‌ی سالانه
- **پیش‌بینی** — موجودی پایان ماه، پتانسیل پس‌انداز، خطر هزینه‌ی بیش از حد، زمان‌بندی اهداف
- **شبیه‌ساز فرضی** — «اگر ۱۵٪ هزینه‌ی رستوران را کاهش دهم، چقدر پس‌انداز می‌کنم؟»
- **بررسی ماهانه** — خلاصه‌ی اجرایی تولیدشده توسط هوش مصنوعی با درآمد، هزینه، امتیاز سلامت، دستاوردها و برنامه‌ی ماه آینده
- **مربیگری هفتگی** — مربیگری شخصی‌سازی‌شده هر دوشنبه ساعت ۰۹:۰۰
- **چندزبانه** — پشتیبانی کامل از فارسی و انگلیسی

### Mini App (وب‌اپ درون تلگرام)

#### اصلی
- **داشبورد** — موجودی حساب‌ها + آخرین تراکنش‌ها در یک نگاه
- **تراکنش‌ها** — لیست صفحه‌بندی‌شده با فیلتر، حذف و فرم افزودن درون‌برنامه‌ای
- **گزارش‌ها** — انتخابگر بازه‌ی زمانی با نمایش بصری تفکیک دسته‌بندی
- **دوستان** — مانده‌های باز و دوستان تسویه‌شده

#### تب هوش مصنوعی (🤖)
- **گفتگوی هوشمند** — رابط مکالمه‌ای با دستیار مالی هوش مصنوعی
- **امتیاز سلامت** — نمایش بصری امتیاز با نوارهای پیشرفت و توضیح هر مؤلفه
- **اهداف** — ایجاد و پیگیری اهداف مالی با نوار پیشرفت و تخمین تکمیل
- **بودجه‌ها** — کارت‌های بودجه با رنگ‌بندی (سبز/زرد/قرمز)، افزودن و حذف با تخصیص دسته‌بندی
- **پیش‌بینی** — درآمد، هزینه و موجودی پیش‌بینی‌شده + زمان‌بندی اهداف
- **اشتراک‌ها** — پرداخت‌های دوره‌ای تشخیص داده‌شده با خلاصه‌ی هزینه‌ی ماهانه/سالانه
- **بینش‌های روزانه** — نکات پیشگیرانه‌ی هوش مصنوعی با قابلیت بازنشانی

## پشته‌ی فناوری

| لایه | فناوری |
|---|---|
| بک‌اند | PHP 8.3 / Laravel 11 |
| SDK ربات | `irazasyed/telegram-bot-sdk` (مبتنی بر Webhook) |
| هوش مصنوعی | DeepSeek `deepseek-reasoner` از طریق `openai-php/client` |
| معماری هوش مصنوعی | مبتنی بر Agent (۹ agent تخصصی + orchestrator) |
| پایگاه داده | MySQL + Eloquent ORM |
| کش / مدیریت وضعیت | Laravel Cache (درایور پایگاه داده) |
| فرانت‌اند | React 18 + TypeScript + Vite |
| کامپوننت‌های UI | `@telegram-apps/telegram-ui` |
| SDK تلگرام | `@telegram-apps/sdk-react` |

## معماری هوش مصنوعی

تمام محاسبات مالی در PHP انجام می‌شود. LLM فقط JSON ساختاریافته دریافت می‌کند و توضیحات، بینش‌ها و توصیه‌ها تولید می‌کند.

```
پیام کاربر
     │
     ▼
AgentOrchestrator  ──── مسیریابی کلیدواژه ────►  Agent تخصصی
                                                       │
                                                ┌──────┴──────┐
                                                │    Tools     │
                                                │  (PHP math)  │
                                                └──────┬───────┘
                                                       │ JSON ساختاریافته
                                                       ▼
                                                  DeepSeek LLM
                                                       │ زبان طبیعی
                                                       ▼
                                                    پاسخ
```

| Agent | مسئولیت |
|---|---|
| `ChatAgent` | پرسش و پاسخ عمومی درباره‌ی مالی کاربر |
| `FinancialCoachAgent` | مربیگری هفتگی شخصی‌سازی‌شده |
| `SpendingAnalyzerAgent` | تحلیل الگوها و عادات هزینه |
| `BudgetAdvisorAgent` | پایش بودجه و توصیه‌ها |
| `ForecastAgent` | پیش‌بینی پایان ماه و تکمیل اهداف |
| `GoalPlannerAgent` | استراتژی‌ها و زمان‌بندی اهداف |
| `SubscriptionAnalyzerAgent` | تشخیص و بررسی پرداخت‌های دوره‌ای |
| `FinancialHealthAgent` | توضیح امتیاز سلامت و نکات بهبود |
| `ReportWriterAgent` | خلاصه‌ی اجرایی ماهانه |

## راه‌اندازی

### پیش‌نیازها
- PHP 8.3 به بالا
- MySQL
- Composer
- Node.js 18 به بالا
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather))
- کلید API دیپ‌سیک ([platform.deepseek.com](https://platform.deepseek.com))
- آدرس HTTPS عمومی (برای توسعه‌ی محلی از ngrok استفاده کنید)

### ۱. نصب وابستگی‌ها

```bash
composer install
```

### ۲. پیکربندی محیط

```bash
cp .env.example .env
php artisan key:generate
```

فایل `.env` را ویرایش کنید:

```env
DB_CONNECTION=mysql
DB_DATABASE=telegram_finance
DB_USERNAME=your_db_user
DB_PASSWORD=your_db_password

TELEGRAM_BOT_TOKEN=123456:ABC-your-token
TELEGRAM_WEBHOOK_URL=https://your-domain.com/webhook/telegram

DEEPSEEK_API_KEY=sk-your-deepseek-key
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-reasoner
```

### ۳. پایگاه داده

```bash
php artisan migrate
php artisan db:seed --class=DefaultCategoriesSeeder
```

### ۴. ثبت Webhook

```bash
curl "https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://your-domain.com/webhook/telegram"
```

برای توسعه‌ی محلی با ngrok:

```bash
ngrok http 8000
# آدرس https://xxxx.ngrok.io را به‌عنوان TELEGRAM_WEBHOOK_URL استفاده کنید
```

### ۵. اجرای سرور

```bash
php artisan serve
```

### ۶. Mini App

```bash
cd mini-app
npm install
npm run build       # خروجی در public/mini-app/
```

آدرس Mini App را در BotFather تنظیم کنید:
Edit Bot ← Edit Menu Button ← `https://your-domain.com/mini-app`

### ۷. زمان‌بند

به crontab اضافه کنید (مدیریت یادآوری‌های تکرارشونده، بینش روزانه، مربیگری هفتگی، بررسی ماهانه):
```
* * * * * cd /path/to/project && php artisan schedule:run >> /dev/null 2>&1
```

| دستور | زمان‌بندی | هدف |
|---|---|---|
| `ai:daily-insights` | روزانه ۰۸:۰۰ | تولید و ارسال بینش‌های پیشگیرانه |
| `ai:weekly-coaching` | دوشنبه‌ها ۰۹:۰۰ | ارسال مربیگری هفتگی شخصی‌سازی‌شده |
| `ai:monthly-review` | اول هر ماه ۱۰:۰۰ | تولید گزارش اجرایی ماهانه |
| `recurring:check` | هر دقیقه | پردازش یادآوری‌های تراکنش تکرارشونده |

### ۸. صف‌کار (Queue Worker)

```bash
php artisan queue:work
```

## API

تمام endpoint‌ها زیر `/api/*` قرار دارند و نیاز به هدر `Authorization: tma <initDataRaw>` دارند (اعتبارسنجی از طریق HMAC-SHA256 با توکن ربات).

### اصلی

| متد | مسیر | توضیح |
|---|---|---|
| GET | `/api/me` | پروفایل کاربر + موجودی کل |
| GET | `/api/accounts` | لیست حساب‌ها |
| POST | `/api/accounts` | ایجاد حساب |
| GET | `/api/categories` | لیست دسته‌بندی‌ها (`?type=expense\|income`) |
| GET | `/api/transactions` | لیست صفحه‌بندی‌شده (`?limit&offset&type&from&to`) |
| POST | `/api/transactions` | ایجاد تراکنش |
| DELETE | `/api/transactions/{id}` | حذف تراکنش |
| GET | `/api/report` | گزارش (`?period=month\|quarter\|year`) |
| GET | `/api/friends` | دوستان با مانده‌ها |
| GET | `/api/friends/{id}/expenses` | هزینه‌های مشترک باز با یک دوست |

### مربی مالی هوشمند

| متد | مسیر | توضیح |
|---|---|---|
| POST | `/api/ai/chat` | گفتگو با هوش مصنوعی (`{message, currency?}`) |
| GET | `/api/ai/insights` | بینش‌های پیشگیرانه‌ی امروز |
| GET | `/api/ai/health-score` | امتیاز سلامت مالی + تفکیک مؤلفه‌ها |
| GET | `/api/ai/subscriptions` | پرداخت‌های دوره‌ای تشخیص داده‌شده |
| GET | `/api/ai/habits` | عادات هزینه‌ای تشخیص داده‌شده |
| POST | `/api/ai/whatif` | شبیه‌سازی فرضی (`{scenario, params, currency?}`) |

### اهداف و بودجه‌ها

| متد | مسیر | توضیح |
|---|---|---|
| GET | `/api/goals` | لیست اهداف مالی |
| POST | `/api/goals` | ایجاد هدف |
| PATCH | `/api/goals/{id}` | به‌روزرسانی پیشرفت یا وضعیت هدف |
| DELETE | `/api/goals/{id}` | حذف هدف |
| GET | `/api/budgets` | لیست بودجه‌ها با میزان استفاده |
| POST | `/api/budgets` | ایجاد بودجه |
| DELETE | `/api/budgets/{id}` | حذف بودجه |
| GET | `/api/forecast` | پیش‌بینی مالی (`?currency=USD`) |
