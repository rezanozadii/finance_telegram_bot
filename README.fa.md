# ربات تلگرام مدیریت مالی شخصی

یک **دستیار مالی هوشمند** کامل، ساخته‌شده به‌صورت ربات تلگرام + Mini App. درآمد و هزینه‌ها را ردیابی کنید، حساب‌ها را مدیریت کنید، هزینه‌ها را با دوستان تقسیم کنید، و از مربی مالی هوش مصنوعی داخلی بهره‌مند شوید — همه درون تلگرام.

> **آخرین نسخه:** v1.2.8 — ترجمه کامل فارسی گزارش‌ها، دسته‌بندی‌ها و امتیاز سلامت مالی؛ سازگاری با Carbon 2.x؛ نمایگر تایپ واقعی تلگرام.

## امکانات

### ربات

#### منوی اصلی (رابط کاربری)
همه کاربران از لحظه تایپ `/start` یک رابط کاربری کامل می‌بینند:
- **کیبورد ثابت پایین صفحه** — ۱۱ دکمه همیشه در دسترس (نیازی به تایپ دستورات نیست)
- **منوی درون‌خطی کامل** — دسترسی به ۱۶ ویژگی با یک ضربه

```
➕ ثبت تراکنش   |  📋 تراکنش‌ها
🏦 حساب‌ها       |  📂 دسته‌بندی‌ها
📊 گزارش ماهانه  |  📈 گزارش سالانه
🎯 اهداف         |  💼 بودجه‌ها
👥 دوستان        |  🔄 تکرارشونده
❤️ سلامت مالی   |  💡 بینش‌های روزانه
🏋️ مربی مالی   |  🤖 چت هوشمند
🔄 اشتراک‌ها    |  🌐 زبان
        📖 آموزش ربات
```

#### دستورات ربات
| دستور | توضیح |
|---|---|
| `/start` | ثبت‌نام و نمایش منوی کامل با همه دکمه‌ها |

| `/accounts` | مشاهده و مدیریت حساب‌ها |
| `/balance` | موجودی دوستان با دکمه افزودن دوست |
| `/transactions` | آخرین تراکنش‌ها با فیلتر هزینه/درآمد/انتقال |
| `/categories` | مرور دسته‌بندی‌های درآمد و هزینه |
| `/report` | گزارش ماهانه/فصلی/سالانه |
| `/budget` | مدیریت بودجه‌ها با نوار پیشرفت |
| `/goals` | پیگیری اهداف مالی |
| `/friends` | دوستان و هزینه‌های مشترک |

| `/recurring` | مدیریت تراکنش‌های تکرارشونده |
| `/health` | امتیاز سلامت مالی (۰ تا ۱۰۰) |
| `/insights` | بینش‌های روزانه هوش مصنوعی |
| `/coach` | مربیگری مالی هفتگی |
| `/ai` | چت هوشمند با دکمه‌های دسترسی سریع |
| `/language` | تغییر زبان (فارسی ↔ انگلیسی) + به‌روزرسانی خودکار ارز |

#### مالی اصلی
- **حساب‌ها** — نقدی، کارت، بانک، کیف پول الکترونیکی، اعتباری با ردیابی لحظه‌ای موجودی
- **دسته‌بندی‌ها** — ۱۴ دسته‌بندی پیش‌فرض + امکان ایجاد سفارشی
- **ثبت دستی تراکنش** — درآمد، هزینه، انتقال با همگام‌سازی موجودی حساب
- **تجزیه‌ی هوش مصنوعی** — پیام طبیعی بفرستید («۵۰ هزار تومان برای ناهار») و DeepSeek تراکنش را استخراج می‌کند
- **یادگیری دسته‌بندی** — اصلاح دسته‌بندی پیشنهادی، قوانین مرتبط با فروشنده را ذخیره می‌کند
- **تراکنش‌های تکرارشونده** — قبوض هفتگی/ماهانه/سالانه با یادآوری و تأیید با یک ضربه
- **دوستان و تقسیم هزینه** — ارسال درخواست دوستی، ثبت هزینه‌های مشترک، ردیابی بدهی، تسویه‌حساب
- **گزارش‌ها** — خلاصه ماهانه/فصلی/سالانه با تفکیک دسته‌بندی و مقایسه با دوره قبل

#### مربی مالی هوشمند
- **`/ai`** — چت طبیعی با دکمه‌های سریع (سلامت، بینش، مربی، اشتراک‌ها، خروج)
- **`/health`** — امتیاز سلامت مالی (۰–۱۰۰) بر اساس ۸ مؤلفه + دکمه مربیگری
- **`/goals`** — تعریف اهداف مالی با تخمین تاریخ تکمیل توسط هوش مصنوعی
- **`/budget`** — سقف هزینه با هشدار ۸۰٪/۹۰٪/۱۰۰٪ و نوارهای پیشرفت رنگی
- **`/insights`** — ۳ بینش پیشگیرانه روزانه ساعت ۰۸:۰۰ با لینک به مربیگری
- **`/coach`** — مربیگری هفتگی شخصی‌سازی‌شده با لینک‌های متقابل
- **شخصیت مالی** — استاد بودجه، پس‌اندازکننده متعادل، هزینه‌کننده سبک زندگی و ...
- **تشخیص اشتراک‌ها** — شناسایی خودکار پرداخت‌های دوره‌ای با هزینه ماهانه/سالانه
- **تشخیص عادات** — شناسایی عادت‌های هزینه‌ای مکرر با فرکانس، میانگین و هزینه سالانه
- **پیش‌بینی** — موجودی پایان ماه، پتانسیل پس‌انداز، خطر هزینه بیش از حد، زمان‌بندی اهداف
- **شبیه‌ساز فرضی** — «اگر ۱۵٪ هزینه رستوران را کاهش دهم، چقدر پس‌انداز می‌کنم؟»
- **بررسی ماهانه** — خلاصه اجرایی تولیدشده توسط هوش مصنوعی اول هر ماه
- **مربیگری هفتگی** — مربیگری شخصی‌سازی‌شده هر دوشنبه ساعت ۰۹:۰۰
- **چندزبانه** — پشتیبانی کامل فارسی و انگلیسی با ارز پیش‌فرض خودکار (ریال / دلار)

### Mini App (وب‌اپ درون تلگرام)

#### صفحات اصلی
- **داشبورد** — موجودی حساب‌ها + آخرین تراکنش‌ها + دکمه ⚙️ تنظیمات + لینک مدیریت حساب‌ها
- **تراکنش‌ها** — لیست صفحه‌بندی‌شده با فیلتر نوع، حذف و فرم افزودن درون‌برنامه‌ای
- **گزارش‌ها** — انتخابگر بازه زمانی با نمایش بصری تفکیک دسته‌بندی
- **دوستان** — مانده‌های باز و دوستان تسویه‌شده

#### مدیریت حساب و دسته‌بندی
- **حساب‌ها** — مدیریت کامل: لیست حساب‌ها، ایجاد جدید (نام، نوع، ارز، موجودی اولیه)
- **دسته‌بندی‌ها** — مرور دسته‌بندی‌های درآمد و هزینه با زیردسته‌ها

#### تنظیمات
- **صفحه تنظیمات** — انتخاب زبان (فارسی/انگلیسی)، اطلاعات پروفایل، نمایش ارز پیش‌فرض
- تغییر زبان به‌صورت خودکار ارز پیش‌فرض را به‌روز می‌کند (فارسی → ریال، انگلیسی → دلار)

#### تب هوش مصنوعی (🤖)
- **گفتگوی هوشمند** — رابط مکالمه‌ای با دستیار مالی هوش مصنوعی
- **امتیاز سلامت** — نمایش بصری امتیاز با نوارهای پیشرفت و توضیح مؤلفه‌ها
- **اهداف** — ایجاد و پیگیری اهداف با نوار پیشرفت و تخمین تکمیل
- **بودجه‌ها** — کارت‌های بودجه رنگی (سبز/زرد/قرمز)، افزودن و حذف
- **پیش‌بینی** — درآمد، هزینه و موجودی پیش‌بینی‌شده + زمان‌بندی اهداف
- **اشتراک‌ها** — پرداخت‌های دوره‌ای تشخیص داده‌شده با خلاصه هزینه ماهانه/سالانه
- **بینش‌های روزانه** — نکات پیشگیرانه روزانه هوش مصنوعی
- **عادات هزینه‌ای** — تحلیل عادت‌ها در سطح فروشنده با هزینه ماهانه/سالانه
- **شبیه‌ساز فرضی** — ۴ سناریو: کاهش دسته‌بندی، افزایش حقوق، پس‌انداز ثابت، لغو اشتراک

## معماری هوش مصنوعی

تمام محاسبات مالی در PHP انجام می‌شود. LLM فقط JSON ساختاریافته دریافت می‌کند و توضیحات، بینش‌ها و توصیه‌ها تولید می‌کند.

```
پیام کاربر
     │
     ▼
AgentOrchestrator  ──── مسیریابی کلیدواژه ────►  Agent تخصصی
                                                       │
                                                ┌──────┴──────┐
                                                │    Tools     │
                                                │  (PHP math)  │
                                                └──────┬───────┘
                                                       │ JSON ساختاریافته
                                                       ▼
                                                  DeepSeek LLM
                                                       │ زبان طبیعی
                                                       ▼
                                                    پاسخ
```

| Agent | مسئولیت |
|---|---|
| `ChatAgent` | پرسش و پاسخ عمومی درباره مالی کاربر |
| `FinancialCoachAgent` | مربیگری هفتگی شخصی‌سازی‌شده |
| `SpendingAnalyzerAgent` | تحلیل الگوها و عادات هزینه |
| `BudgetAdvisorAgent` | پایش بودجه و توصیه‌ها |
| `ForecastAgent` | پیش‌بینی پایان ماه و تکمیل اهداف |
| `GoalPlannerAgent` | استراتژی‌ها و زمان‌بندی اهداف |
| `SubscriptionAnalyzerAgent` | تشخیص و بررسی پرداخت‌های دوره‌ای |
| `FinancialHealthAgent` | توضیح امتیاز سلامت و نکات بهبود |
| `ReportWriterAgent` | خلاصه اجرایی ماهانه |

## راه‌اندازی

### پیش‌نیازها
- PHP 8.3 به بالا
- MySQL
- Composer
- Node.js 18 به بالا
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather))
- کلید API دیپ‌سیک ([platform.deepseek.com](https://platform.deepseek.com))
- آدرس HTTPS عمومی (برای توسعه محلی از ngrok استفاده کنید)

### ۱. نصب وابستگی‌ها

```bash
composer install
```

### ۲. پیکربندی محیط

```bash
cp .env.example .env
php artisan key:generate
```

فایل `.env` را ویرایش کنید:

```env
DB_CONNECTION=mysql
DB_DATABASE=telegram_finance
DB_USERNAME=your_db_user
DB_PASSWORD=your_db_password

TELEGRAM_BOT_TOKEN=123456:ABC-your-token
TELEGRAM_WEBHOOK_URL=https://your-domain.com/webhook/telegram

DEEPSEEK_API_KEY=sk-your-deepseek-key
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-reasoner
```

### ۳. پایگاه داده

```bash
php artisan migrate
php artisan db:seed --class=DefaultCategoriesSeeder
```

### ۴. ثبت Webhook

```bash
curl "https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://your-domain.com/webhook/telegram"
```

برای توسعه محلی با ngrok:

```bash
ngrok http 8000
# آدرس https://xxxx.ngrok.io را به‌عنوان TELEGRAM_WEBHOOK_URL استفاده کنید
```

### ۵. اجرای سرور

```bash
php artisan serve
```

### ۶. Mini App

```bash
cd mini-app
npm install
npm run build       # خروجی در public/mini-app/
```

آدرس Mini App را در BotFather تنظیم کنید:
Edit Bot ← Edit Menu Button ← `https://your-domain.com/mini-app`

### ۷. زمان‌بند

```
* * * * * cd /path/to/project && php artisan schedule:run >> /dev/null 2>&1
```

| دستور | زمان‌بندی | هدف |
|---|---|---|
| `ai:daily-insights` | روزانه ۰۸:۰۰ | تولید و ارسال بینش‌های پیشگیرانه |
| `ai:weekly-coaching` | دوشنبه‌ها ۰۹:۰۰ | مربیگری هفتگی شخصی‌سازی‌شده |
| `ai:monthly-review` | اول هر ماه ۱۰:۰۰ | گزارش اجرایی ماهانه |
| `recurring:check` | هر دقیقه | پردازش یادآوری‌های تراکنش تکرارشونده |

### ۸. صف‌کار

```bash
php artisan queue:work
```

## API

تمام endpoint‌ها زیر `/api/*` قرار دارند و نیاز به هدر `Authorization: tma <initDataRaw>` دارند (HMAC-SHA256 با توکن ربات).

### کاربر

| متد | مسیر | توضیح |
|---|---|---|
| GET | `/api/me` | پروفایل کاربر + موجودی کل |
| PATCH | `/api/me` | به‌روزرسانی زبان و/یا ارز پیش‌فرض |

### اصلی

| متد | مسیر | توضیح |
|---|---|---|
| GET | `/api/accounts` | لیست حساب‌ها |
| POST | `/api/accounts` | ایجاد حساب |
| GET | `/api/categories` | لیست دسته‌بندی‌ها (`?type=expense\|income`) |
| GET | `/api/transactions` | لیست صفحه‌بندی‌شده (`?limit&offset&type&from&to`) |
| POST | `/api/transactions` | ایجاد تراکنش |
| DELETE | `/api/transactions/{id}` | حذف تراکنش |
| GET | `/api/report` | گزارش (`?period=month\|quarter\|year`) |
| GET | `/api/friends` | دوستان با مانده‌ها |
| GET | `/api/friends/{id}/expenses` | هزینه‌های مشترک با یک دوست |

### مربی مالی هوشمند

| متد | مسیر | توضیح |
|---|---|---|
| POST | `/api/ai/chat` | گفتگو با هوش مصنوعی (`{message, currency?}`) |
| GET | `/api/ai/insights` | بینش‌های امروز |
| GET | `/api/ai/health-score` | امتیاز سلامت مالی + تفکیک مؤلفه‌ها |
| GET | `/api/ai/subscriptions` | پرداخت‌های دوره‌ای تشخیص داده‌شده |
| GET | `/api/ai/habits` | عادات هزینه‌ای (`?currency=`) |
| POST | `/api/ai/whatif` | شبیه‌سازی فرضی (`{scenario, params, currency?}`) |

**سناریوهای شبیه‌ساز:** `reduce_category`، `salary_increase`، `save_fixed`، `cancel_subscription`

### اهداف و بودجه‌ها

| متد | مسیر | توضیح |
|---|---|---|
| GET | `/api/goals` | لیست اهداف مالی |
| POST | `/api/goals` | ایجاد هدف |
| PATCH | `/api/goals/{id}` | به‌روزرسانی پیشرفت یا وضعیت |
| DELETE | `/api/goals/{id}` | حذف هدف |
| GET | `/api/budgets` | لیست بودجه‌ها با میزان استفاده |
| POST | `/api/budgets` | ایجاد بودجه |
| DELETE | `/api/budgets/{id}` | حذف بودجه |
| GET | `/api/forecast` | پیش‌بینی مالی (`?currency=USD`) |
